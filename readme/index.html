最友好的Canvas客户端

易

Better Canvas目前可以一直保持登录，省下了官方版Canvas每次需要先等待加载登录页面，并输入密码的时间。

在官方版的Canvas中，进入某个课程(Course)或在课程间切换需要鼠标移上顶栏，等待其加载并展开后才能选择具体课程。Better Canvas将所有课程如书签般列在左侧，直接点击即可切换；而在选择课程后，顶部的标签栏使通知、作业、讨论等各项功能的切换依旧方便。

在通知(Announcement)，讨论(Discussion)等其他界面，官方版Canvas会先展示出一张通知列表，列表中的每一项只有第一行内容的预览(并没有卵用)。当你想查看某条通知的全部内容时，居然还需要等浏览器打开一个新窗口，并重新加载所有界面元素(但你只是想看通知的后半部分文字而已)；当你看完通知，后退到通知列表时，居然还要等他全部刷新！Better Canvas永不打开新窗口，所有列表项目均能直接点击展开和折叠内容。此功能适用于通知、讨论、作业(Assignments)和页面(Pages)。

Better Canvas同时有其他贴心的功能，如回到顶部按钮，在页面内容较多时可以快速回到顶部的工具栏。如果你对Better Canvas感兴趣，请在页面最后留下你心目中Canvas该有的功能。

「你自己的Canvas」

美

Canvas作为每天都会使用的应用，想必你多少对它灰+红的页面有些审美疲劳(呃，它其实不需要审美)。Better Canvas在致力提高它的易用性的同时，也希望拥有丰富的个性化界面。目前的Better Canvas使用更漂亮的字体和更大的字号，并运用透明、阴影等CSS效果，给你更好的使用体验。同时它支持更换背景图片(有人吐槽过为什么我做的东西全都可以换背景图片)，而且只要图片好，视觉效果就不会差。日后会根据情况加入更换主题等功能。

快，不需说明

轻

Canvas是一个功能繁杂的平台，它80%的功能可能平时都不会被用到。Better Canvas专注于同学们平时常用的功能，并希望能够因此更轻便和灵敏。由于所有操作只在一个网页中进行，它节省了许多不必要的新页面渲染时间。同时，它最大化地利用API所返回的信息，避免重复请求。

由于某些原因，Better Canvas中的每项操作仍然需要一点时间等待学校服务器的反应。如果你在使用OSX上的Safari(和我一样)，可以尝试下载一份纯js的版本，该版本的反应速度更要快上几倍。
